📊 TÉCNICAS DISPONIBLES
═══════════════════════════════════════════════════════════════════════

1️⃣  ÁRBOLES DE DECISIÓN
    ┌─ Teoría:    Particionamiento jerárquico, criterios Gini/Entropía
    ├─ Supuestos: Datos representativos, etiquetado de calidad
    ├─ Práctica:  Predicción de biodiversidad marina
    ├─ Código:    100 líneas (scikit-learn, visualización)
    └─ Ejemplo:   Clasifica hábitats por T°, salinidad, pH, nitratos

2️⃣  ANÁLISIS PCA
    ┌─ Teoría:    Componentes principales, reducción dimensional
    ├─ Supuestos: Variables continuas, estandarización, correlaciones
    ├─ Práctica:  Análisis de 8 variables oceanográficas
    ├─ Código:    80 líneas (scree plot, biplot)
    └─ Ejemplo:   Proyecta 20 estaciones a 2 componentes

3️⃣  K-MEANS CLUSTERING
    ┌─ Teoría:    Iteración de asignación-actualización
    ├─ Supuestos: Clusters esféricos, tamaño similar
    ├─ Práctica:  Segmentación de hábitats marinos
    ├─ Código:    90 líneas (método del codo, silhouette)
    └─ Ejemplo:   Agrupa 50 sitios en 3 hábitats

4️⃣  ANOVA
    ┌─ Teoría:    Contraste de hipótesis F, post-hoc
    ├─ Supuestos: Normalidad, homocedasticidad, independencia
    ├─ Práctica:  Comparación entre 3 regiones costeras
    ├─ Código:    85 líneas (Tukey HSD, eta-squared)
    └─ Ejemplo:   Compara biodiversidad en región 1,2,3


💻 REQUISITOS

  ✓ Navegador moderno (Chrome, Firefox, Safari, Edge)

🛠 PERSONALIZACIÓN
═══════════════════════════════════════════════════════════════════════

Para añadir nuevo caso:
  1. Abre cases.json
  2. Copia el objeto de un caso existente
  3. Modifica fields: id, title, source, theory, assumptions, practice, code, tryit
  4. Guarda y recarga index.html

Para cambiar estilos:
  1. Edita styles.css
  2. Variables en :root {} controlan colores
  3. Recarga la página

Para cambiar texto de interfaz:
  1. Edita en index.html (títulos, textos fijos)
  2. Usa variables en cases.json para contenido dinámico
