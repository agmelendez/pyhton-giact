Teorema del Límite Central y Teoría de Colas

Convergencia a normalidad y modelado de colas exponenciales

El TLC establece que el promedio de n muestras independientes e idénticamente distribuidas converge a una distribución normal cuando n→∞. En teoría de colas, un sistema M/M/1 modela procesos con llegadas Poisson (tasa λ) y servicio exponencial (tasa μ).

Convergencia a distribución normal con media μ y desviación estándar σ/√n
Válido para n≥30 incluso con distribuciones originales no normales
Cola M/M/1: llegadas Poisson, servicio exponencial, 1 servidor
Requisito fundamental: λ < μ para estabilidad del sistema

Z = (X̄ - μ) / (σ/√n); ρ = λ/μ; W = λ/(μ(μ-λ))

PY-02

Análisis de Varianza (ANOVA) con Pruebas Post-Hoc

Comparación de medias entre k grupos mediante prueba F

ANOVA es un método estadístico que compara k medias simultáneamente mediante el cociente F = MSentre / MSdentro. Rechaza hipótesis nula si hay diferencias significativas entre grupos.

Generalización de prueba t para dos grupos a k > 2 grupos
Controla tasa de error tipo I global mediante ANOVA en lugar de k pruebas t
F = MSentre / MSdentro sigue distribución F(k − 1, n − k) bajo H0
Requiere pruebas post-hoc (Tukey HSD) para identificar pares que difieren

F = SSentre / (k − 1)
─────────────
SSdentro / (n − k)

η² = SSentre / SStotal

Principios de Diseño Experimental

Estructura y análisis de experimentos con control y replicación

Experimento controlado asigna aleatoriamente unidades a grupos de tratamiento vs control, replica observaciones en cada grupo, y utiliza ANOVA para comparar efectos. La aleatorización minimiza sesgos por variables de confusión.

Aleatorización: asignación aleatoria de unidades a tratamientos/control
Control: grupo control recibe placebo o no-tratamiento como referencia
Replicación: múltiples unidades por grupo para estimar variabilidad
ANOVA y pruebas post-hoc para análisis estadístico formal

Efecto = Media_tratamiento - Media_control; η² = SS_tratamiento / SS_total

PY-04

Análisis de Componentes Principales (PCA) — Fundamentos

Transformación ortogonal para reducción dimensional

PCA es una transformación ortogonal que reexpresa variables originales como combinaciones lineales no correlacionadas (componentes principales) ordenadas decrecientemente por varianza explicada. Reduce dimensionalidad preservando máxima información.

Componentes = combinaciones lineales ortogonales de variables originales
Ordenados por varianza decreciente: PC1 > PC2 > ... > PCp
Loadings = pesos que combinan variables originales en cada PC
Scores = proyección de observaciones en espacio de componentes principales

PC_i = w_i1·X_1 + w_i2·X_2 + ... + w_ip·X_p; Var(PC_1) ≥ Var(PC_2) ≥ ... ≥ Var(PC_p)

PY-06

Análisis de Conglomerados K-Means

Particionamiento iterativo en k clusters homogéneos

K-Means es algoritmo de clustering que asigna n observaciones a k clusters (definidos a priori) minimizando suma de cuadrados intra-cluster (SSE). Itera entre asignación a centroides y actualización de centroides hasta convergencia.

Objetivo: minimizar SSE = Σ||x_i - centroide_k||² dentro de clusters
Requiere estandarización de variables para comparabilidad de escalas
Número k puede definirse a priori o determinarse por método del codo
Converge localmente; aleatorización múltiple de iniciales mejora solución

SSE = Σ_k Σ_{i∈cluster_k} ||x_i - μ_k||²; μ_k = promedio de observaciones en cluster k

PY-07

Regresión Múltiple con Diagnóstico de Multicolinealidad

Modelado de Y basado en múltiples predictores con diagnóstico de violaciones

Regresión múltiple modela relación lineal Y = β₀ + β₁X₁ + ... + βₙXₙ + ε estimada por Mínimos Cuadrados Ordinarios (OLS). Supuestos LINE (Linealidad, Independencia, Normalidad, Equivariance) y bajo VIF (ausencia multicolinealidad) son críticos.

OLS minimiza suma de cuadrados de residuos Σ(y_i - ŷ_i)²
Supuestos LINE más diagnóstico de multicolinealidad (VIF) aseguran validez de inferencia
VIF = 1/(1-R²ⱼ) mide inflación de varianza de βⱼ por correlación con otros predictores
Observaciones influyentes (Cook's D>1) pueden distorsionar regresión

Ŷ = β₀ + β₁X₁ + ... + βₙXₙ; R² = 1 - SS_residual/SS_total; VIF_j = 1/(1-R²_j)

PY-08

Series de Tiempo y Pronóstico ARIMA

Componentes autorregresivos, integración y media móvil

ARIMA(p,d,q) combina Autorregresión (AR=p lags de valores pasados), Integración (I=d diferenciaciones), y Media Móvil (MA=q lags de errores pasados). Modela series estacionarias y las hace estacionarias mediante diferenciación.

AR(p): regresión de y_t sobre p valores pasados y_{t-1}, ..., y_{t-p}
I(d): d diferenciaciones para alcanzar estacionariedad; d=1 diferencia simple, d=2 segunda diferencia
MA(q): regresión de y_t sobre q errores pasados ε_{t-1}, ..., ε_{t-q}
Flexibilidad: puede modelar tendencias (I), autocorrelación (AR), e innovaciones (MA)

ARIMA: Δ^d y_t = β₀ + Σᵢ φᵢ Δ^d y_{t-i} + Σⱼ θⱼ ε_{t-j} + ε_t

Pronóstico de Lluvia Mensual con SARIMAX

Modelado de series con patrones estacionales y variables externas

SARIMAX(p,d,q)(P,D,Q,s) extiende ARIMA con componentes estacionales P,D,Q de periodo s (e.g., s=12 para datos mensuales anuales). Incluye variables exógenas (regresores externos). Ideal para lluvia monthly con ciclo anual fuerte.

P,D,Q son órdenes estacionales; s=12 para anual, s=4 para trimestral, s=7 para semanal
D diferenciación estacional (s pasos atrás); usualmente D=0 o 1
Variables exógenas: predictores externos (e.g., temperatura, presión) incluidos como regresores
Descomposición previa en tendencia, estacional, residuo ayuda a especificar órdenes

SARIMAX: Δ^d Δ_s^D y_t = β₀ + Σ φ_i y_{t-i} + Σ Φ_j y_{t-sj} + Σ θ_k ε_{t-k} + Σ Θ_l ε_{t-sl} + Σ γ_r X_{t-r} + ε_t

PY-10

Árboles de Decisión para Gestión Marino-Costera

Método no-paramétrico para clasificación y decisión

Árbol de decisión es modelo que particiona recursivamente el espacio de características seleccionando en cada nodo la variable y umbral que maximiza ganancia de información (medida por Gini o entropía). Genera reglas interpretables para predicción y toma de decisiones.

Particionamiento recursivo: divide datos en subgrupos cada vez más puros
Criterio de split: Gini impurity o información (entropía); busca minimizar impureza en hojas
No-paramétrico: sin supuestos distribucionales; maneja relaciones no-lineales naturalmente
Interpretabilidad: reglas de decisión en forma de árbol fáciles de comunicar

Gini = 1 - Σ(p_k)²; Ganancia de Información = Gini(padre) - Σ(n_hijo/n_padre)·Gini(hijo)
